# Wazuh SIEM Deployment

## Overview

This project documents the deployment of Wazuh SIEM within my Proxmox homelab environment.

## Objectives

- Centralize security logs
- Monitor Linux and Windows systems
- Learn threat detection workflows
- Gain hands-on SIEM experience

## Environment

### Wazuh Manager

Ubuntu Server

### Agents

- Ubuntu Linux
- Windows 11

## Features

- File Integrity Monitoring
- Vulnerability Detection
- Threat Hunting
- MITRE ATT&CK Mapping
