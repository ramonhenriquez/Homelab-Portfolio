# Self-Hosted AI Platform

## Overview

This project documents the deployment of a locally hosted AI platform for experimenting with large language models (LLMs) and private AI workflows.

---

## Objectives

* Explore self-hosted AI technologies
* Maintain data privacy
* Evaluate local model performance
* Develop AI-assisted workflows

---

## Environment

### Platform

Odysseus

### Models

* Local LLMs
* Self-hosted inference

---

## Features

* Local AI Chat Interface
* Private Data Processing
* Self-Hosted Deployment
* Custom Model Testing

---

## Security Considerations

* No external API dependency
* Local processing
* Controlled access
* Private infrastructure
