# Jellyfin Media Server

## Overview

This project documents the deployment of a self-hosted Jellyfin media server connected to centralized NAS storage.

The service provides media streaming capabilities while demonstrating storage integration and service administration.

---

## Objectives

* Learn self-hosted service deployment
* Integrate NAS storage
* Manage user access
* Understand media infrastructure

---

## Architecture

Jellyfin LXC

↓

NAS Storage

↓

Media Libraries
